Step 1) How Your DNS Servers are set in your Network ?

Depending on your company policy, there can be various sources that can be used for setting your DNS. Easiest way to findout is to check whether you have DHCP Enabled in your computer. If that is the case, you must locate the DHCP Server which can be on a router, a firewall or some server itself, login and change the DNS addresses as in step 2.

If that is not the case, just change your DNS server from Adapter Settings from your personal computer or from Wi-Fi Settings from your mobile phone. You can further check out instructions for your computer from https://www.comodo.com/secure-dns/switch/computer.html

Step 1) The Default Policy and Profiles

Upon creation of your network as in Add Network Tutorial, Dome Shield automatically applies a Default Security Policy to your network blocking Malware, Phising and Spyware sources. This Default Policy can not be deleted but you can create:

Category Rules from Configure Category Rules Create Category Rule helps you to define rules for blocking or allowing domain categories.

Security Rules from Configure Security Rules Create Security Rule helps you to define security profiles.

B/W Lists from Configure B/W Lists Create B/W List helps you to create custom black and whitelists.

Step 2) Create Policy

Following creation of Profiles you must go to Configure Policy and click Add Policy.

Select the network which you want to set a policy for.

Select a Security Rule. Dome Shield enforces you to select at least the Default Security Rule.

Select a Category Rule. If you don't want to have a Category Rule, you can leave it empty as long as you have a Security Rule selected.

You can add Domain B/W Lists and by clicking Enable Only B/W Mode you can disable Security and Category Rules.
